### Available endpoints:

| Endpoint                                      | Purpose        |
| --------------------------------------------- | -------------- |
| `/`                                           | Dashboard      |
| `/api/alerts`                                 | Latest alerts  |
| `/api/alerts/1`                               | Single alert   |
| `/api/response?status=yes&location=Back Gate` | Confirm threat |
| `/api/response?status=no`                     | False alarm    |
| `/api/police/Back Gate`                       | Police lookup  |
| `/health`                                     | Health check   |
